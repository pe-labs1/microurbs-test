# Microburbs Test - Property Investment Explorer

A modern property dashboard designed to help investors and homebuyers identify high-potential properties using data-driven insights.

## Business Domain

In the competitive Australian property market, making informed decisions requires more than just a price tag. This application bridges the gap between property listings and deep market intelligence.

**Key Value Propositions:**
*   **Liveability Scores**: Instantly assess the lifestyle value of a property based on proximity to schools, parks, transport, and cafes.
*   **Investment Health**: Visualize market trends, rental yields, and capital growth potential to identify high-return assets.
*   **Development Activity**: Track nearby Development Applications (DAs) to spot up-and-coming areas with increasing infrastructure and value.

## Features

### 1. Interactive Property Map
Explore properties geographically with color-coded markers indicating their Liveability Score.
*   **Green**: High Liveability (80+)
*   **Amber**: Medium Liveability (50-79)
*   **Red**: Low Liveability (<50)

### 2. Deep Dive Property Insights
Click on any property to view a comprehensive report:
*   **Property Features**: At-a-glance view of bedrooms, bathrooms, land size, and more.
*   **Liveability Breakdown**: Detailed scoring for convenience, lifestyle, and family friendliness.
*   **Growth Signals**: Analysis of local development activity and market temperature.

## Tech Stack

*   **Framework**: Next.js 15+ (App Router)
*   **Styling**: Tailwind CSS
*   **Mapping**: React Leaflet & OpenStreetMap
*   **Icons**: Lucide React

## Getting Started

1.  Navigate to the project directory:
    ```bash
    cd microurbs-test
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Run the development server:
    ```bash
    npm run dev
    ```

4.  Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
